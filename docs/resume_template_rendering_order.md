# Resume Template Section Rendering Order

## Single Column Templates

Applies to: Classic, Corporate, Finance, Charcoal Glow

| Order | Section         |
|-------|-----------------|
| 1     | Contact Info    |
| 2     | Profile         |
| 3     | Skills          |
| 4     | Work Experience |
| 5     | Projects        |
| 6     | Achievements    |
| 7     | Education       |
| 8     | Certifications  |
| 9     | Languages       |

## Two Column Templates

Applies to: Mercury, Steady Form

### Narrow Column (Sidebar)

| Order | Section      |
|-------|--------------|
| 1     | Contact Info |
| 2     | Languages    |
| 3     | Skills       |

### Wide Column (Main)

| Order | Section         |
|-------|-----------------|
| 1     | Profile         |
| 2     | Work Experience |
| 3     | Projects        |
| 4     | Achievements    |
| 5     | Education       |
| 6     | Certifications  |

> **Note:** The header (name + profession) is rendered either as a full-width bar above both columns (Steady Form) or at the top of the main column (Mercury).

## Rules

- Sections with no data must not render.
- Section order is strict and must not vary.
- Photos/images are not supported in any template.
